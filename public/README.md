# 静的ファイルの配置場所

このフォルダには、アプリケーションで使用する静的ファイル（動画、画像など）を配置してください。

## 動画ファイルの配置方法

1. **動画ファイルを配置**
   - このフォルダ（`public`）に動画ファイル（`.mp4`, `.webm`など）を配置してください
   - 例：`public/video.mp4`

2. **ファイル名を確認**
   - `src/App.tsx`の`handleSend`関数内で、`setVideoSrc('/video.mp4')`と設定されています
   - 動画ファイルの名前が異なる場合は、この部分を修正してください

3. **動画フォーマット**
   - 推奨：MP4（H.264）
   - その他：WebM、OGG

## サポートされているファイル形式

- **動画**: `.mp4`, `.webm`, `.ogg`
- **画像**: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`

## 注意事項

- 動画ファイルが大きい場合は、読み込み時間が長くなる可能性があります
- 動画ファイルがない場合は、`src/App.tsx`の`setVideoSrc('/video.mp4')`をコメントアウトしてください
