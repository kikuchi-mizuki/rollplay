## はじめに

このシステムは、SNS動画制作会社の営業トレーニングを自動化するためのロールプレイングツールです。

### 主な機能
- ✅ AIが相談者役として自然に会話
- ✅ 音声認識・音声合成による実践的な練習
- ✅ 6つのシナリオ（1次面談〜追加営業）
- ✅ SNS動画営業特化のAI講評（4軸評価）
- ✅ 履歴閲覧・CSV出力

## 初回ログイン

### 1. アクセス

ブラウザで以下のURLにアクセスします：
```
https://your-app-url.vercel.app
```

### 2. Google認証

「Googleでログイン」ボタンをクリックし、Googleアカウントで認証します。

### 3. 店舗コード入力

初回登録時に、店舗コードの入力を求められます。

```
店舗コード例: STORE_001
```

店舗コードは、店舗管理者または本部管理者から発行されます。

### 4. プロフィール情報入力

- 表示名（例: 山田太郎）
- 店舗コード（自動入力済み）

入力後、「登録」ボタンをクリックします。

## シナリオの選択

ヘッダーの「シナリオ選択」ドロップダウンから、練習したいシナリオを選びます。

### 6つのシナリオ

| シナリオ | 特徴 | 難易度 |
|---------|-----|-------|
| **1次面談** | 初回接触・関係構築 | ★☆☆ |
| **1.5次面談** | 情報交換・信頼構築 | ★★☆ |
| **2次面談** | 提案・検討 | ★★☆ |
| **3次面談** | クロージング | ★★★ |
| **キックオフMTG** | プロジェクト開始 | ★☆☆ |
| **追加営業** | 既存顧客への提案 | ★★★ |

### シナリオの選び方

- **初心者**: 1次面談から順番に練習
- **中級者**: 2次面談、3次面談で提案力を磨く
- **上級者**: 追加営業で新たな価値提案を練習

## ロープレの実施

### 1. 会話の開始

シナリオを選択したら、**あなたから最初に話しかけます**。

**例:**
```
「本日はお時間いただきありがとうございます」
「こんにちは、〇〇と申します」
```

### 2. 入力方法

#### テキスト入力
- 入力エリアにメッセージを入力
- 「送信📤」ボタンをクリック

#### 音声入力（推奨）
- 「録音🎤」ボタンをクリック
- マイクに向かって話す（最低2秒）
- 自動的に文字起こしされて送信

### 3. AIの応答

AIが相談者役として応答します。

- **音声**: 自動的に音声で返答
- **字幕**: 画面に字幕表示
- **動画**: リップシンクアバター（オプション）

### 4. 会話の継続

自然な会話を続けます。

**ヒアリング → 共感 → 提案 → クロージング** の流れを意識しましょう。

### 5. 会話の終了

- 十分な会話（5〜10往復程度）ができたら、「講評📊」ボタンをクリック

## AI講評の見方

### 評価項目（4軸）

| 評価軸 | 配点 | 意味 |
|-------|-----|-----|
| **ヒアリング力** | 25% | 相談者の課題を引き出せたか |
| **共感力** | 25% | 不安・懸念に寄り添えたか |
| **提案力** | 25% | 具体的な解決策を提示できたか |
| **信頼構築力** | 25% | 自然に次のステップに誘導できたか |

### スコアの見方

```
⭐⭐⭐⭐⭐ 5点 - 非常に優秀
⭐⭐⭐⭐   4点 - 優秀
⭐⭐⭐     3点 - 良い
⭐⭐       2点 - 改善の余地あり
⭐         1点 - 要改善
```

### 講評の活用

1. **良かった点**: 継続して強化する
2. **改善点**: 次回の練習で意識する
3. **アドバイス**: 具体的な改善策を実践する

## 履歴の確認

### 履歴ページ

ヘッダーの「履歴」リンクをクリックすると、過去の練習履歴を確認できます。

### 表示内容

- 練習日時
- シナリオ
- 平均スコア
- 各評価軸のスコア

### フィルター機能

- シナリオ別にフィルタリング
- スコア推移グラフ（実装予定）

### CSV出力

「CSV出力」ボタンをクリックすると、Excel形式でデータをダウンロードできます。

**用途:**
- 個人の成長記録
- 店舗での研修管理
- 本部への報告資料

## トラブルシューティング

### 音声が聞こえない

**原因:** ブラウザの自動再生ブロック

**解決策:**
1. ページ内をクリックして、音声を有効化
2. ブラウザの設定で自動再生を許可

### マイクが使えない

**原因:** マイクの権限が許可されていない

**解決策:**
1. ブラウザのアドレスバーの🔒アイコンをクリック
2. 「マイク」を「許可」に変更
3. ページをリロード

### AIの返答が遅い

**原因:** ネットワーク接続が遅い、またはサーバーが混雑している

**解決策:**
1. ネットワーク接続を確認
2. しばらく待つ（通常5〜10秒程度）
3. タイムアウトした場合は再送信

### プロフィール取得エラー

**原因:** データベース接続の問題

**解決策:**
1. ページをリロード
2. ログアウト→再ログイン
3. 解決しない場合は管理者に連絡

### 店舗コードが無効

**原因:** 店舗コードが存在しない、または入力ミス

**解決策:**
1. 店舗コードを再確認（大文字・小文字、スペースに注意）
2. 店舗管理者に正しいコードを確認

## よくある質問（FAQ）

### Q1. 何回でも練習できますか？

A. はい、制限なく何度でも練習できます。

### Q2. 他の人の練習履歴は見られますか？

A. いいえ、自分の履歴のみ閲覧可能です（管理者を除く）。

### Q3. モバイルでも使えますか？

A. はい、スマートフォン・タブレットでも利用可能です。

### Q4. オフラインでも使えますか？

A. いいえ、インターネット接続が必要です。

### Q5. 評価は誰が行いますか？

A. AIが自動的に評価します。人間の講師は不要です。
